## 如何使用

### 1. 查看演示结果（已运行）
```bash
python demo.py
```

### 2. 实际使用（需要网络连接）
```bash
python research_buddy.py
```
这将抓取真实的大学网站：https://nxy.hunau.edu.cn/xygk/szdw_9365/

### 3. 自定义筛选条件

编辑 `research_buddy.py` 文件，修改筛选条件：
```python
criteria = {
    'required_titles': ['教授', '副教授'],     # 职称要求
    'research_keywords': ['你关心的关键词'],   # 研究关键词
    'required_degree': '博士'                  # 学历要求（可选）
}
```

## 功能特点

✅ **自动化抓取** - 自动访问教师列表和个人主页  
✅ **智能提取** - 提取姓名、职称、研究方向、联系方式等  
✅ **灵活筛选** - 支持多种条件组合筛选  
✅ **数据导出** - JSON格式保存，方便后续分析  
✅ **演示模式** - 无需网络即可测试功能  
✅ **中文支持** - 完美支持中文网页和信息
